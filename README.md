## 一些使用小技巧

用了这么久,我也总结了几个省钱的小窍门:

**1. 优先选USPS Cubic定价**

如果你的包裹重量在1-20磅之间,体积又不算太大(大概鞋盒那么大或者更小),强烈建议选USPS Priority Mail Cubic。这个定价方式是按体积算的,不是按重量,往往比普通的Priority Mail便宜很多。

Goshippo会自动帮你计算,看看是Cubic划算还是按重量划算,你只要选最便宜的那个就行。

**2. 用免费的USPS包装盒**

USPS有很多免费的Flat Rate包装盒和信封,你可以在USPS网站上订,他们会免费寄给你。如果你的东西正好能塞进这些固定尺寸的盒子,有时候会比用自己的盒子更便宜。

Goshippo在选择包装类型的时候,会把Flat Rate选项列出来,你可以对比一下。

**3. 别忘了包含的免费保险**

很多人不知道,USPS的Priority Mail和Priority Mail Express都自带100美元的保险。如果你的包裹价值不高,就不用额外买保险了,省下这笔钱。

**4. 批量打印标签**

如果你一次要寄很多包裹,Goshippo有批量导入功能。你可以把所有订单信息整理成CSV文件,一次性上传,然后批量打印标签。比一个一个手动操作快多了。

**5. 地址验证很重要**

Goshippo有地址验证功能,美国地址验证是免费的。虽然看起来是个小功能,但能帮你避免因为地址错误导致的退件,省时省力还省钱。
